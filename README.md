# Spomassas Acari - Formulário de Pedidos

Este repositório contém o formulário online para fazer pedidos de massas do Spomassas Acari.  
O formulário envia os pedidos diretamente para o WhatsApp do restaurante.

## Como usar

1. Abra o site hospedado no GitHub Pages: https://camila100788.github.io/teste_spomassas/
2. Selecione sua proteína, massa, molho, ingredientes, finalização e bebidas.
3. Preencha seu nome e endereço.
4. Clique em **"Enviar Pedido no WhatsApp"**.
5. O pedido será enviado diretamente para o WhatsApp do restaurante.

## Estrutura do projeto

- `index.html` → Página principal com o formulário
- `style.css` → (Opcional) Estilo do formulário
- `script.js` → (Opcional) Lógica do envio para WhatsApp
- `assets/` → Imagens e ícones usados no site
